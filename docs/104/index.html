2008-03-09 13:59 folder [前端 技术]  

     HTML 5 还增加了一些纯语义性的块级元素： 
aside figure  dialog
我在文章和书中一直使用前两个元素。第三个元素我不经常用，它主要用于书面文本。 
aside
